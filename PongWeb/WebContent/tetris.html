Tetris HTML5

Instrucciones

Para PC:
  • Empezar / pausar el juego: pulsa Intro o doble Click.
  • Mover pieza: o A/D/S o mover el ratón.
  • Rotar pieza: o W o hacer Click con el ratón.

Instrucciones

Para Móvil/Tablet:
  • Empezar / pausar el juego: hacer doble tap.
  • Mover pieza: Mover el dedo por la pantalla.
  • Rotar pieza: Hacer tap sobre la pantalla.
